# ЛАБОРАТОРНАЯ РАБОТА №1 ДЕ ПОЙ НИКОЛАЙ ИДБ-24-14
====================================================

В этой лабораторной работе разработано представление системы управления книжным магазином процедурным способом. По запуску программы открывается меню, где можно воспользоваться различными инструменами по типу: просмотр книг в магазине, поиск книг по параметрам, удаление или продажа книги, просмотр рабочих и т.д. В работе используется 5 классов и разработаны кастомные классы исключений.
